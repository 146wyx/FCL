# Fun Craft Launcher (FCL)

一个基于 .NET 10 开发的现代化 Minecraft Java 版启动器，采用 WPF 技术构建，提供流畅的用户体验和丰富的功能支持。

![启动器主界面](screenshots/main.png)

## ✨ 功能特性

### 🎮 游戏启动
- **正版/离线登录支持** - 支持 Microsoft 正版登录和离线模式
- **自动补全文件** - 启动前自动检查并下载缺失的游戏文件
- **版本隔离** - 支持版本独立的游戏目录，避免存档冲突
- **快速启动** - 记住上次选择的版本，一键启动游戏

### 📥 版本下载
- **官方版本下载** - 支持下载所有官方 Minecraft 版本
- **Mod 加载器支持** - 支持 Forge、NeoForge、Fabric、Quilt 等主流加载器
- **多线程下载** - 使用 BMCLAPI 镜像，高速下载游戏文件
- **自动安装** - 下载完成后自动安装并配置

### ⚙️ 版本管理
- **版本设置** - 为每个版本单独配置 Java、内存、窗口等参数
- **版本删除** - 一键删除不需要的版本，释放磁盘空间
- **启动统计** - 记录每个版本的启动次数和最后启动时间

### 🔧 全局设置
- **Java 配置** - 自动查找或手动指定 Java 路径，调整内存分配
- **游戏目录** - 自定义默认游戏目录
- **下载设置** - 选择下载源，调整并发下载数
- **外观主题** - 深色/浅色主题切换，支持多语言

### 🛡️ 其他特性
- **文件完整性检查** - 启动前验证游戏文件完整性
- **资源索引更新** - 后台自动更新资源文件索引
- **日志系统** - 详细的日志记录，方便排查问题
- **现代化 UI** - 采用 Fluent Design 设计风格，支持动画效果

## 🚀 快速开始

1. **首次启动** - 启动器会自动创建默认的游戏目录
2. **下载游戏** - 点击"下载"按钮，选择想要的 Minecraft 版本
3. **登录账号** - 支持 Microsoft 正版登录或离线模式
4. **启动游戏** - 选择版本，点击"启动游戏"按钮

## 📝 使用说明

### 正版登录
1. 在启动页面点击"正版登录"按钮
2. 在浏览器中完成 Microsoft 账号授权
3. 授权成功后自动返回启动器

### 版本设置
1. 选择要配置的游戏版本
2. 点击"版本设置"按钮
3. 调整 Java 路径、内存、窗口大小等参数
4. 点击"保存"应用设置

### 安装 Mod 加载器
1. 在下载页面选择"Mod 加载器"选项卡
2. 选择要安装的加载器类型（Forge/Fabric 等）
3. 选择对应的 Minecraft 版本
4. 点击下载并等待安装完成

## 🔧 系统要求

- **操作系统**: Windows 10/11 (64位)
- **运行环境**: .NET 10 Runtime
- **内存**: 建议 4GB 以上
- **磁盘空间**: 根据游戏版本不同，需要 1-2GB 空间
